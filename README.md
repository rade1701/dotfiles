# Athenyx Dotfiles
**Ubuntu WSL · Sway · Waybar Tokyo Night · Fish**

## Install
```bash
rm -rf ~/.dotfiles
bash <(curl -fsSL https://raw.githubusercontent.com/rade1701/dotfiles/refs/heads/main/install.sh)
```

## Stack
| App | Command |
|-----|---------|
| Sway WM | `sway` |
| Brave browser | `brave` |
| Bitwarden GUI | `bitwarden` |
| Bitwarden CLI | `bw` |
| Claude Desktop | `claude-desktop` |
| Claude Code CLI | `claude` or `cc` |
| OpenAI CLI | `openai` |

## Keybindings (Mod = Alt in WSLg)
| Key | Action |
|-----|--------|
| Alt+Return | Terminal |
| Alt+D | Fuzzel launcher |
| Alt+B | Brave |
| Alt+H/J/K/L | Focus |
| Alt+Shift+H/J/K/L | Move |
| Alt+F | Fullscreen |
| Alt+Shift+F | Float |
| Alt+R | Resize mode |
| Alt+1-5 | Workspace |
| Alt+P | Screenshot |
| Alt+Shift+C | Reload Sway |
| Alt+Shift+Q | Exit |
